# Workers on TON — Telegram Bot

## Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Настройка

Отредактируй `config.py`:
- `BOT_TOKEN` — токен бота от @BotFather
- `TON_WALLET` — твой TON кошелёк для приёма депозитов
- `USDT_WALLET` — твой USDT (TRC20) кошелёк
- `LOG_CHAT_ID` — Telegram ID администратора (для заявок на вывод)

### 3. Запуск

```bash
python main.py
```

## Функционал

| Модуль | Возможности |
|--------|------------|
| Магазин | 10 типов рабочих (1–5000 TON), покупка, подтверждение |
| Ферма | 10 уровней фермы с бонусом до +75%, бонусы за достижения |
| Доход | Автоначисление, ручной сбор, учёт бустов и фарм-бонуса |
| Бусты | 5 временных (до +150%) + 4 постоянных (до +25%) |
| Рефералы | 5 уровней, 7%/3%/2%/1%/0.5%, бонусы за действия друзей |
| Вывод | FSM-диалог, 3 скорости, лимиты, уведомление админа |
| Достижения | 4 достижения с наградами до 200 TON |
| Топ | Топ по заработку и по рефералам |
| Ежедневный бонус | Серия до 30 дней, бонус до 5 TON |
| Админка | Одобрение/отклонение выводов, начисление баланса |

## Добавление депозитов

Депозиты обрабатываются вручную или через внешний скрипт.
Для зачисления используй команду `/addbalance <user_id> <amount>` в боте.

Для автоматизации подключи TON API (например, TonCenter) и добавь вебхук-обработчик,
который при получении транзакции вызывает `add_balance(user_id, amount)` из `database/db.py`.
